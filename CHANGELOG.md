# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-12

### Added
- Initial release
- Secure ENV-based authentication with JWT
- Server management (add, list, delete)
- Browser-based SSH terminal with xterm.js
- WebSocket tunneling via Socket.io
- AES-256 encryption for private keys
- Connection validation before saving servers
- Docker and Docker Compose support
- Modern dark-themed UI
- File upload for SSH keys
- Custom delete confirmation modal
